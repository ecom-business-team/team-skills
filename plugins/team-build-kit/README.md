# Team Build Kit

The three-gate build lifecycle for building tools you can trust — without being an engineer.

**Owner:** Zachary Blake

**The gates:**

| Command | When |
|---|---|
| `/memo` | You have an idea worth building — prove it before designing |
| `/prd` | The idea is worth it — now design it so it can be built one-shot |
| `/build` | The design is solid — implement it |
| `/ship` | It works — but others will rely on it, or it touches money or real data |
| `/quick-fix` | Something small broke, or a tiny tweak — no ceremony |
| `/new-workspace` | Set up a tidy home for your work before any of the above |

**The idea:** *complexity is the enemy. If you can't explain it simply, it's probably too complicated.*

Install with `/plugins install team-build-kit`.
